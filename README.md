# chatter

...have you ever wondered which cryptocurrencies are people talking about today?

## Development

**Dependencies**

- Node.js >=18 and yarn
- PostgreSQL >= 15

**Backend**

```sh
cd backend
yarn
yarn dev:db-migrate
yarn dev:server
```

Backend API is now accessible on <http://localhost:4001/>

**Frontend**

```sh
cd frontend
yarn
yarn dev:server
```

App is now accessible on <http://localhost:4000/>
